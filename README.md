# Real-Time-Driver-Safety-System-ADAS-
This project builds a real-time AI system that acts like a co-pilot, monitoring the driver and road simultaneously to prevent accidents before they happen.

Built a real-time multi-modal ADAS system integrating lane departure warning, object detection (YOLOv8), driver drowsiness monitoring (MediaPipe), and forward collision warning using Python and OpenCV, deployed as a live Streamlit dashboard.
